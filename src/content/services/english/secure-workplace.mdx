## Security that fits a small business — without the enterprise complexity [.text-h4]

Most small businesses are more exposed than they realise. Weak passwords, unmanaged devices, shared accounts, and no conditional access policies are common — and attackers know it.

The Secure Workplace package identifies and addresses your real security risks on Microsoft 365, using the same controls that large organisations rely on, configured to be practical for a team of 5–25 people.

## What's included [.text-h4]

Every Secure Workplace engagement includes:

- Identity and access review — user accounts, admin roles, guest access, shared mailboxes
- MFA assessment and enforcement for all users (including legacy protocol blocking)
- Microsoft Secure Score review with prioritised remediation plan
- Microsoft Intune enrolment for company devices (Windows, Mac, iOS, Android)
- Device compliance policies aligned to your risk profile
- Conditional access policy configuration (location, device compliance, risk-based)
- Microsoft Defender for Business basic configuration
- Security baseline report with before/after Secure Score comparison

This package is recommended as an add-on to Digital Workplace Foundation, or as a standalone review for businesses already on Microsoft 365.
